Decentralized Auditable Affiliate Website Skeleton

Features

  • Log Visitors
  • Track forms submitted
  • Track links clicked
  • Setup email marketing funnel
  • single page application

Pages

  • Offer
  • Thank you

Contracts

  • Website - logs visitors, user clicks, and form submits; tracks who is the owner
  • Captcha - receives answer and url of new image and new answer
  • Audit - pays people to run a program that checks the blocks and tests their validity by adding at least three secrets in each block; returns website stats; returns users to be paid
  • Auction - tracks highest bidder; transfers contract ownership; checks audit was performed; pays all parties invovled their percentages
